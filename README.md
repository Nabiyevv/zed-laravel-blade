# zed-laravel-blade

Laravel Blade templating language support for [Zed](https://zed.dev).

> [!Important]
> Add this in your zed setting to automatically select Blade mode for `.blade.php` files instead of php.
> ```json
> "file_types": {
>   "Blade": ["*.blade.php"]
> }
> ```
